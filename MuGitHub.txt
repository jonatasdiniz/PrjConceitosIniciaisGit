Git & Github

Dados do desenvolver
1. Configurando a pessoa, desenvolvedor que irá registrar os commits
git config --global user.name ""
retorna o nome do usuário

2. Verificar o email do usuário cadastrado no Github
git config --global user.email ""
retorna email cadastrado para os commits

Diferença em Global, Local, System - serve somente para o repositório local

Configuração Padrão Branch inicial MAIN
3 . git config --global init.defaultbranch main

* Movendo Branch de Master para Main
git branch --move main
